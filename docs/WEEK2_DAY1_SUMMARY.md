**问题描述：**
- `npm install` 执行时间过长
- 多次尝试都超时或被 kill
- `node_modules` 未成功安装

**可能原因：**
1. 环境权限问题
2. 网络连接问题
3. 依赖包版本冲突

**解决方案：**
1. 在有网络环境下的机器上重新安装
2. 使用 CI/CD 自动安装
3. 使用 `npm ci` 替代 `npm install`
