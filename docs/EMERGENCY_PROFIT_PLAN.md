## ⚙️ 运营策略

### 资源使用原则
1. **严格控本** - 每次调用必须有明确目的
2. **收入导向** - 不产生收入的活动一律暂停
3. **快速验证** - 用最小成本验证需求
4. **现金为王** - 优先考虑能立即产生现金的业务

### 营销策略
1. **精准定位** - 找真正需要 AI 技术的客户
2. **案例驱动** - 用后端代码作为技术证明
3. **口碑传播** - 超越预期，获得推荐
4. **价格灵活** - 初期可以低价获取案例
