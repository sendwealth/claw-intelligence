## 生命周期概述

### 生命周期阶段图

```
入职期                成长期                 成熟期                  扩展期
(0-7 天)             (7-30 天)             (30-90 天)              (90 天+)
  │                    │                     │                      │
  ├─ 欢迎与启动         ├─ 功能深化            ├─ 业务对齐              ├─ 季度战略会议
  ├─ 产品激活           ├─ 价值验证            ├─ ROI分析              ├─ 价值报告
  ├─ 功能探索           ├─ 关系巩固            ├─ 高级功能              ├─ 新功能体验
  └─ 价值确认           └─ 习惯养成            └─ 续费准备              └─ 增购机会

目标：7天激活          目标：30天留存         目标：90天续费           目标：年度增购
```

### 生命周期核心指标

| 阶段 | 核心目标 | 关键指标 | 目标值 |
|------|---------|---------|--------|
| 入职期 | 激活使用 | 7天激活率 | > 85% |
| 成长期 | 深度使用 | 30天留存率 | > 90% |
| 成熟期 | 价值确认 | 90天续费率 | > 80% |
| 扩展期 | 增长扩展 | 年度增购率 | > 50% |

## 入职期（Onboarding）

### 阶段定义

**时间范围：** 0-7 天（从签约完成开始）

**核心目标：** 帮助客户快速上手，7天内激活使用并产生首次价值

**关键成功因素：**
- 快速响应（< 4小时）
- 清晰的引导流程
- 即时问题解决
- 早期价值感知

---

### Day 1：欢迎与启动

#### 核心活动

**1. 欢迎邮件发送**
- [ ] 发送欢迎邮件（包含）

- [ ] 联系客户确认时间

- [ ] 在客户成功平台创建客户档案

- [ ] 发送数字欢迎礼包
