## 快速开始

### 第一次使用

1. **注册账号**
   - 访问 CLAW.AI 网站
   - 点击"注册"按钮
   - 填写用户名、密码和邮箱
   - 完成注册

2. **登录系统**
   - 使用注册的用户名和密码登录
   - 登录后进入主界面

3. **创建知识库**
   - 点击"知识库"菜单
   - 点击"创建知识库"按钮
   - 填写知识库名称和描述
   - 点击"创建"

4. **上传文档**
   - 进入知识库详情页
   - 点击"上传文档"按钮
   - 选择要上传的文件
   - 等待上传和向量化完成

5. **开始提问**
   - 进入对话界面
   - 选择要使用的知识库
   - 输入问题
   - 获取基于知识库的回答

## 用户注册与登录

### 注册账号

**步骤：**
1. 点击页面右上角的"注册"按钮
2. 填写注册信息：
   - **用户名**：3-20个字符，字母、数字、下划线
   - **密码**：至少8个字符，建议包含大小写字母、数字和特殊字符
   - **邮箱**（可选）：用于找回密码
3. 阅读并同意服务条款
4. 点击"注册"按钮

**密码安全建议：**
- ✅ 使用至少12个字符
- ✅ 包含大小写字母
- ✅ 包含数字
- ✅ 包含特殊字符（!@#$%^&*）
- ❌ 不要使用常见密码（如：123456、password）
- ❌ 不要使用个人信息（如：生日、手机号）

### 登录系统

**步骤：**
1. 点击页面右上角的"登录"按钮
2. 输入用户名和密码
3. 点击"登录"按钮

**忘记密码？**
- 点击"忘记密码"链接
- 输入注册时使用的邮箱
- 查收重置密码的邮件
- 按照邮件中的指引重置密码

## 知识库管理

### 创建知识库

**步骤：**
1. 点击左侧菜单的"知识库"
2. 点击"创建知识库"按钮
3. 填写知识库信息：
   - **名称**：知识库的名称（必填）
   - **描述**：简要描述知识库的内容（可选）
   - **Embedding 模型**：选择文本向量化模型（默认即可）
4. 点击"创建"按钮

**知识库使用场景：**
- 📚 技术文档库 - 存储技术文档、API 文档
- 📖 产品手册 - 存储产品说明、用户手册
- 💼 业务文档 - 存储业务流程、规章制度
- 🎓 学习资料 - 存储学习笔记、课程资料
- 🏢 公司知识 - 存储公司政策、FAQ

### 查看知识库列表

**步骤：**
1. 点击左侧菜单的"知识库"
2. 可以看到所有知识库的列表
3. 每个知识库显示：
   - 名称
   - 文档数量
   - 创建时间

### 编辑知识库

**步骤：**
1. 点击知识库名称进入详情页
2. 点击"编辑"按钮
3. 修改知识库信息
4. 点击"保存"按钮

### 删除知识库

**⚠️ 注意：** 删除知识库会同时删除所有文档和向量索引，此操作不可恢复！

**步骤：**
1. 点击知识库名称进入详情页
2. 点击"删除"按钮
3. 确认删除操作

## 文档上传

### 支持的文件格式

| 格式 | 说明 | 大小限制 |
|------|------|----------|
| **TXT** | 纯文本文件 | 10MB |
| **MD / Markdown** | Markdown 格式 | 10MB |
| **PDF** | PDF 文档 | 10MB |

### 上传文档（文件）

**步骤：**
1. 进入知识库详情页
2. 点击"上传文档"按钮
3. 选择要上传的文件
4. （可选）修改文档标题
5. 点击"上传"按钮
6. 等待上传和向量化完成

**上传进度：**
- 文档上传完成后，系统会自动：
  1. 解析文档内容
  2. 将文档分块
  3. 生成文本向量
  4. 存储到向量数据库

**注意事项：**
- ✅ 上传前确保文档内容完整
- ✅ 文件名建议使用有意义的名称
- ✅ 大文件建议压缩后上传
- ❌ 不要上传包含敏感信息的文档
- ❌ 不要上传加密的 PDF 文件

### 创建文档（直接提供内容）

**步骤：**
1. 进入知识库详情页
2. 点击"创建文档"按钮
3. 填写文档信息：
   - **标题**：文档标题（必填）
   - **内容**：文档内容（必填）
   - **文件类型**：txt 或 markdown（必填）
4. 点击"创建"按钮

**适用场景：**
- 复制粘贴网络上的内容
- 从其他编辑器直接复制内容
- 快速创建测试文档

### 查看文档列表

**步骤：**
1. 进入知识库详情页
2. 可以看到该知识库的所有文档
3. 每个文档显示：
   - 标题
   - 文件类型
   - 分块数量
   - 创建时间

### 删除文档

**步骤：**
1. 进入知识库详情页
2. 找到要删除的文档
3. 点击"删除"按钮
4. 确认删除操作

## AI 对话与 RAG 查询

### 创建对话

**步骤：**
1. 点击左侧菜单的"对话"
2. 点击"创建对话"按钮
3. 填写对话标题
4. 选择要使用的知识库（可选）
5. 点击"创建"按钮

### 对话界面说明

**界面布局：**
```
┌─────────────────────────────────────────────┐
│  [对话历史侧边栏]  [知识库面板]  [聊天窗口]   │
├─────────────────┬───────────────────────────┤
│                 │                           │
│  对话列表       │  聊天消息区域              │
│  - 对话1       │  ┌─────────────────────┐   │
│  - 对话2       │  │ 消息1 (用户)       │   │
│  - 对话3       │  ├─────────────────────┤   │
│                 │  │ 消息2 (AI)         │   │
│  [新建对话]     │  ├─────────────────────┤   │
│                 │  │ 消息3 (用户)       │   │
│                 │  ├─────────────────────┤   │
│                 │  │ 消息4 (AI)         │   │
│                 │  └─────────────────────┘   │
│                 │                           │
│                 │  [输入框] [发送按钮]       │
└─────────────────┴───────────────────────────┘
```

**功能区：**
- **对话历史侧边栏**：查看和管理历史对话
- **知识库面板**：选择要使用的知识库
- **聊天窗口**：查看消息历史和发送新消息

### 发送消息

**步骤：**
1. 在输入框中输入问题
2. （可选）选择要使用的知识库
3. 点击"发送"按钮或按 Enter 键
4. 等待 AI 生成回答

**AI 回答说明：**
- AI 会根据选择的知识库生成回答
- 回答中会引用来源文档
- 来源显示在回答下方

### RAG 查询特点

**什么是 RAG？**
- RAG（Retrieval-Augmented Generation）是一种结合检索和生成的技术
- AI 会先从知识库中检索相关内容
- 然后基于检索到的内容生成回答

**RAG 的优势：**
- ✅ 回答基于知识库，更准确
- ✅ 可以引用来源，可追溯
- ✅ 知识库可更新，回答会同步更新
- ✅ 支持企业私有数据

### 多知识库查询

**步骤：**
1. 在知识库面板中勾选多个知识库
2. 发送消息
3. AI 会综合多个知识库的内容生成回答

**使用场景：**
- 同时查询技术文档和产品手册
- 综合多个部门的业务文档
- 对比不同来源的信息

## 常见问题

### 登录问题

**Q: 忘记密码怎么办？**
A: 点击登录页的"忘记密码"链接，输入注册时使用的邮箱，系统会发送重置密码的邮件。

**Q: 登录后提示"Token 已过期"怎么办？**
A: Token 有效期为 60 分钟，过期后需要重新登录。

### 知识库问题

**Q: 可以上传多大文件？**
A: 单个文件最大支持 10MB。

**Q: 支持哪些文件格式？**
A: 支持 TXT、Markdown 和 PDF 格式。

**Q: 上传文档后多久可以开始查询？**
A: 通常在 1-5 秒内，取决于文档大小。

**Q: 删除知识库后可以恢复吗？**
A: 不可以，删除操作不可恢复，请谨慎操作。

### 对话问题

**Q: AI 回答不准确怎么办？**
A: 可以尝试：
1. 更具体地描述问题
2. 选择更相关的知识库
3. 检查知识库中是否有相关文档

**Q: 可以同时使用多个知识库吗？**
A: 可以，在知识库面板中勾选多个知识库即可。

**Q: 对话记录会保存多久？**
A: 对话记录会永久保存，直到手动删除。

### 性能问题

**Q: 响应速度慢怎么办？**
A: 可以尝试：
1. 检查网络连接
2. 清理浏览器缓存
3. 联系技术支持

**Q: 大知识库查询速度慢？**
A: 大知识库查询可能需要 3-10 秒，这是正常现象。

## 最佳实践

### 知识库管理

**1. 合理组织知识库**
- 按主题分类创建多个知识库
- 避免单个知识库包含过多无关文档
- 使用清晰的命名规则

**2. 文档命名规范**
- 使用有意义的文件名
- 包含日期或版本号（如：`API文档_v1.0.md`）
- 避免使用中文文件名（可能造成兼容性问题）

**3. 文档质量**
- 确保文档内容完整、准确
- 使用清晰的标题和结构
- 避免重复内容

**4. 定期更新**
- 及时更新过时的文档
- 删除不再需要的文档
- 添加新文档时标注版本

### 对话技巧

**1. 问题表述**
- 尽量具体，避免模糊问题
- ❌ "怎么用这个产品？"
- ✅ "如何在 CLAW.AI 中创建知识库？"

**2. 上下文信息**
- 提供必要的上下文信息
- ❌ "这个问题怎么解决？"
- ✅ "上传 PDF 文档时提示解析错误，该怎么解决？"

**3. 多轮对话**
- 可以基于上一轮的回答继续提问
- 保持对话连贯性

### 安全建议

**1. 数据安全**
- 不要上传包含敏感信息的文档
- 定期备份重要的对话记录
- 不要与他人共享账号

**2. 密码安全**
- 使用强密码
- 定期更换密码
- 不要在多个网站使用相同密码

**3. 内容合规**
- 不要上传违法、有害内容
- 尊重知识产权
- 不要上传他人文档
