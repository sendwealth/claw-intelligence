## 📡 API 接口（13 个）

**认证模块：**
- POST /auth/register - 用户注册
- POST /auth/login - 用户登录
- GET /auth/me - 获取用户信息

**对话模块：**
- POST /conversations - 创建对话
- GET /conversations - 查询对话列表
- DELETE /conversations/{id} - 删除对话
- POST /conversations/{id}/chat - 聊天接口

**消息模块：**
- GET /messages/{conversation_id} - 获取消息历史

**WebSocket：**
- WS /ws/chat - 实时聊天
- WS /ws/online - 在线用户
