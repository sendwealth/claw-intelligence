### 3. Issues 作为咨询入口

**创建 Issues 模板：**

```yaml
name: 咨询需求
description: 描述你的项目需求，我们会尽快回复
title: "[咨询] 简短描述"
body:
  - type: textarea
    id: description
    attributes:
      label: 需求描述
      description: 请详细描述你的项目需求
      placeholder: 例如：我想开发一个基于 GLM-4 的智能客服系统...
    validations:
      required: true

  - type: dropdown
    id: service_type
    attributes:
      label: 服务类型
      options:
        - 技术咨询（¥200/小时）
        - 项目咨询（¥800-2000/项目）
        - 后端开发（¥3000 起/项目）
        - 源码购买
        - 月度顾问（¥2000/月）
    validations:
      required: true

  - type: textarea
    id: budget
    attributes:
      label: 预算范围（可选）
      description: 如果方便，请提供预算范围

  - type: input
    id: contact
    attributes:
      label: 联系方式（可选）
      description: 邮箱或微信
      placeholder: your@email.com
```

**预期效果：**
- 规范化咨询流程
- 自动收集需求信息
- 提高转化率
