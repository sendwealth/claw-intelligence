## 🔧 部署方式

### 方式 A：通过 IP 地址直接部署（推荐，快速）

**优点：**
- 不需要域名解析
- 立即可用
- 适合备案审核

**步骤：**
1. 通过 IP 地址访问服务器
2. 上传网站文件
3. 配置 Web 服务器
4. 通过 IP 地址访问网站（http://47.xx.xx.xx）

### 方式 B：通过域名部署

**条件：**
- 域名已经解析到服务器 IP
- 域名 DNS 解析生效

**步骤：**
1. 配置域名 DNS 解析
2. 等待解析生效（5-30 分钟）
3. 通过域名访问网站

## 📋 部署步骤（等你提供信息后执行）

### 第 1 步：连接服务器
```bash
ssh root@<你的服务器IP>
```

### 第 2 步：安装 Web 服务器
```bash
# 安装 Nginx
apt update && apt install nginx -y
# 或
yum install nginx -y
```

### 第 3 步：上传网站文件
- 创建网站目录
- 上传 HTML、CSS 文件
- 配置权限

### 第 4 步：配置 Nginx
- 设置网站根目录
- 配置虚拟主机
- 启用配置

### 第 5 步：测试访问
- 通过 IP 地址访问
- 检查所有页面
- 确认内容正确
